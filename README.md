### 🏷️ Rule
- 매주 `금요일` `10:30 ~ 12:00` (비대면)
- 중간고사 / 기말고사 전 주는 휴식
- 일주일에 4문제씩 고정으로 풀기 
  - 코드 리뷰로 스터디원의 풀이 확인
- 알자디 시간에는 프로그래머스 2단계 최신순 같이 풀기 
  - 화면 공유하여 30분~1시간
